Это простой калькулятор для крафта в игре Lineage 2
Для начала запустите скрипт calc.py
После чего вам будет предложено ввести название крафта. Это может быть название самого предмета.
Затем необходимо ввести количество видов ингридиентов.
Затем название, количество и цену каждого ингридиента.
Потом ввести цену крафта и количество крафтов.
После чего программа выдаст общее количество необходимой адены, а также сохранит эти данные в файле statistika.txt
